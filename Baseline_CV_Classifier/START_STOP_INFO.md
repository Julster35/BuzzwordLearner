# LinkedIn CV Classifier - Start/Stop Skripte

## App starten

Doppelklick auf:
```
start_app.bat
```

Oder im Terminal:
```bash
.\start_app.bat
```

Die App öffnet sich automatisch im Browser unter **http://localhost:8501**

## App stoppen

**Option 1:** Im Terminal-Fenster `Strg+C` drücken

**Option 2:** Doppelklick auf:
```
stop_app.bat
```

Oder im Terminal:
```bash
.\stop_app.bat
```

## Hinweise

- Die Skripte müssen aus dem `Baseline_CV_Classifier` Ordner ausgeführt werden
- `start_app.bat` startet die Streamlit-App
- `stop_app.bat` beendet alle laufenden Streamlit-Prozesse
- Nach dem Stoppen kann die App mit `start_app.bat` neu gestartet werden
